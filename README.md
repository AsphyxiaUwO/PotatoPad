# Bill of Materials/Components
- 1x 0.91" 128x32 OLED (SSD1306)
- 4x Cherry MX Switch
- 4x DSA Keycaps
- 1x Seeed XIAO RP2040
- 4x Heatset Inserts
- 4x M3x16mm screws
